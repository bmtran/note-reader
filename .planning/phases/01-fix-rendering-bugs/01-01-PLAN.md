# Plan 1-1: Render Hints as VexFlow Notes

## Objective
Replace raw SVG `<ellipse>` hint rendering with proper VexFlow `StaveNote` instances in a shared Voice. Hints must align to actual staff lines, use quarter/half note duration, and render at 30–40% opacity.

## Context
- Current code draws hints as raw SVG ellipses with hardcoded Y math (lines 321–359)
- Target note already uses Voice + Formatter (lines 304–310)
- User decided: single shared Voice, hints as quarter/half notes, target at right end
- VF4 has no `setX()` — positioning must come from Formatter

## Tasks

### Task 1: Build shared Voice with hints + target

<read_first>
- index.html (full file — see drawGrandStaff lines 267–366)
- .planning/phases/01-fix-rendering-bugs/01-CONTEXT.md
</read_first>

<action>
Refactor `drawGrandStaff()` to:
1. Create array of `StaveNote` objects for all hint keys + target key
2. Hint notes: duration `'q'` (quarter), style `{ fillStyle: '#bbbbbb', strokeStyle: '#bbbbbb' }` + set opacity via SVG attribute or context filter
3. Target note: duration `'w'` (whole), no gray style
4. Add all notes to single `Voice({num_beats: N, beat_value: 4})` where N = total note count
5. Call `new Formatter().joinVoices([voice]).format([voice], width - 20)`
6. Draw voice on correct stave (treble or bass)
7. Remove all raw SVG ellipse creation code (lines 321–359)
</action>

<acceptance_criteria>
- `drawGrandStaff()` contains no `createElementNS('ellipse')` calls
- Voice created with `num_beats` equal to total hint count + 1 (target)
- Hint notes use `setStyle({ fillStyle: '#bbbbbb', strokeStyle: '#bbbbbb' })`
- Target note uses duration `'w'` and no gray style
- `voice.draw(context, stave)` called once for the shared voice
- Hints render as actual note heads on correct staff lines (not floating ellipses)
</acceptance_criteria>

### Task 2: Apply 30–40% opacity to hint notes

<read_first>
- index.html (drawGrandStaff function after Task 1 changes)
</read_first>

<action>
For each hint note (not target):
1. After `setStyle()`, add `note.attrs.opacity = '0.35'` or set SVG group opacity
2. If VF4 doesn't support opacity via attrs, use context `save()`/`setGlobalAlpha(0.35)`/`restore()` around hint note draw
3. Alternative: use `setStyle({ fillStyle: 'rgba(187,187,187,0.35)', strokeStyle: 'rgba(187,187,187,0.35)' })` if VF4 accepts rgba
</action>

<acceptance_criteria>
- Hint note heads, stems, and flags all render at ~35% opacity
- Target note remains fully opaque
- Opacity value is 0.30–0.40 range
</acceptance_criteria>
